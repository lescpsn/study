                Golang开发环境搭建
1 windows下的Golang开发环境搭建
    1.1 下载windows对应版本的Golang安装包（当前为version-1.7）

    1.2 采用默认安装方式，如有要求请指定Golang的安装路径，其它一路回车法

    1.3 添加GOPATH环境变量（windows下如何添加环境变量，自行百度），项目工程目录必须放在%GOPATH%/src目录下
        如：
        %GOPATH%/src/username/projectname1
        %GOPATH%/src/username/projectname2
        如何设置GOPATH的值，请根据自己的目录规划习惯了

    1.4 将%GOPATH%/bin路径追加到%PATH%环境变量中

2 linux（ubuntu）下的Golang开发环境搭建
  2.1 网络方式直接安装golang
      apt-get install golang
  2.2 配置GOPATH环境变量
      bash shell:
      vim ~/.bashrc
      export GOPATH=~/projects/golang

      fish shell:
      vim ~/.config/fish/config.fish (若没有，直接创建该文件)
      set -x GOPATH ~/Projects/lescpsn/golang
  2.3 GOPATH下创建三个必须的目录
      mkdir $GOPATH/src
      mkdir $GOPATH/bin
      mkdir $GOPATH/pkg

3. 安装Golang常用工具
   godef: 代码跳转
   go get -u github.com/rogpeppe/godef

   gocode: 代码补全
   go get -u github.com/nsf/gocode
