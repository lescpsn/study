* Windows
1 github上下载spacemacs仓储到windows的HOME目录下，并命名为.emacs.d(必须是.emacs.d)
  下载spacemacs到windows的HOME目录下（我的是C:\Users\carhj\AppData\Roaming）
  git clone https://github.com/syl20bnr/spacemacs.git C:\Users\carhj\AppData\Roaming\.emacs.d

2 启动emacs
2.1 选择emacs模式
On the planet Emacs in the Holy control tower(emacs)
Among the stars aboard the Evil flagship(vim)

2.2 选择spacemacs
The standard distribution, recommended(spacemacs)
A minimalist distribution that you can build on (spacemacs-base)

2.3 选择A heavy one but full-featured(helm)




3 修改 C:\Users\carhj\AppData\Roaming\.spacemacs
  dotspacemacs-elpa-timeout 60


1 修改LPA的仓储源
  .spacemacs文件中的defun dotspacemacs/user-init () 里修改或添加
(setq configuration-layer--elpa-archives
    '(("melpa-cn" . "http://elpa.emacs-china.org/melpa/")
      ("org-cn"   . "http://elpa.emacs-china.org/org/")
      ("gnu-cn"   . "http://elpa.emacs-china.org/gnu/")))
 )

1 修改默认字体大小
 dotspacemacs-default-font '("Source Code Pro"
                             :size 18
                             :weight normal
                             :width normal
                             :powerline-scale 1.2)




2  插件仓储链接超时，修改超时时间60秒
dotspacemacs-elpa-https t
;; Maximum allowed time in seconds to contact an ELPA repository.
dotspacemacs-elpa-timeout 60


3  设置字体 source code font
    下载 source code 字体：https://github.com/adobe-fonts/source-code-pro/downloads

	TTF将TTF下所有文件拷贝到c:/windows/font

zlib安装
http://gnuwin32.sourceforge.net/packages/zlib.htm 下载windows的zlib
(Complete package, except sources)
一路回车安装，将安装后的bin添加到PATH环境变量中去


aspell的安装：
aspell下载全部语言版本
http://aspell.net/win32/
Aspell-0-50-3-3-Setup.exe
Aspell-en-0.50-2-3.exe


4 Error: (file-error "Searching for program" "no such file or directory" "diff")
  需要安装diff工具（Linux系统因为自带diff，Windows自己安装一个）
  下载https://sourceforge.net/projects/kdiff3/files/kdiff3/0.9.98/
   安装KDiff3-64bit-Setup_0.9.98-2.exe，并执行文件bin目录配置到环境变量%PATH%中去
