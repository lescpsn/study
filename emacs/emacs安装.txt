Windows下的emacs安装
1 gnu网站http://ftp.gnu.org/gnu/emacs/windows下载windows对应的emacs版本
  如：我下载二进制bin版本，直接解压就可以使用
  http://ftp.gnu.org/gnu/emacs/windows/emacs-24.5-bin-i686-mingw32.zip

2 目录规划，根据个人习惯，做好目录组织结构。
  如：我将E:/Local定位为类似Linux下的home家目录，后面准备用来存放.emacsd文件夹的位置

3 将emacs-24.5-bin-i686-mingw32.zip解压到E:/Local/目录下， 并改名为emacs，
  E:/Local/emacs

4 emacs启动加载配置文件的两种方式
  windows下家目录HOME，相当与linux下用户主目录的概念。
  一般为C:\Users\当前登陆的用户\AppData\Roaming

  4.1 以家目录下的.emacs文件加载emacs启动配置信息
      当配置信息比较负责繁多的时候，这种方式不利于规划，可能emacs后面会取消这种方式，
      目前还在过渡阶段，仍然可以使用
      4.1.1 编辑 %HOME%/.emacs
            编写个人的emacs配置信息
      4.1.2 如果4.1.1的方式不利于目录规划，可以采用自定义的.emacs文件的位置方式
            %HOME%/.emacs文件中指定个人的.emacs文件的位置，如：E:/Local/.emacs
            (load-file "E:/Local/.emacs")
            编写"E:/Local/.emacs"文件，加入如下配置
            (setenv "HOME" "E:/local")
            (setenv "PATH" "E:/local")
            ;;set the default file path
            (setq default-directory "~/")
            (add-to-list 'load-path "~/emacs/site-lisp")
            之后，你就可以在E:/Local/.emacs中添加你的个人配置信息

  4.2 以家目录下的.emacs.d目录加载emacs启动配置信息
      配置%HOME%/.emacs.d文件夹，可以直接从github上下载开源配置好的。
      C:\Users\当前登陆的用户\AppData\Roaming\.emacs.d
      git clone XXXXX  %HOME%/.emacs.d
